# 产品概览与全局约定

## 1. 产品定位

**Ideal Cafe** —— 独立精品咖啡馆点单小程序。

一句话描述：让顾客在微信里完成看菜单、下单、等取餐的全流程，让店员高效处理订单，让管理员轻松管理商品。

品牌调性关键词：简约、精致、品质感。

**单店模型**：本产品仅服务于一家门店，不涉及连锁门店选择。

## 2. 目标用户与角色定义

| 角色 | 代号 | 访问平台 | 描述 | 核心场景 |
|------|------|---------|------|---------|
| 消费者 | C | 微信小程序 | 来店消费的顾客 | 浏览菜单 → 下单 → 等取餐 → 查看历史订单 |
| 店员 | S | 网页版 | 咖啡馆工作人员 | 查看新订单 → 接单 → 制作 → 通知取餐 |
| 管理员 | A | 网页版 | 咖啡馆经营者 | 管理商品/分类 → 配置门店信息 → 查看订单数据 |

**平台与角色**：小程序仅服务消费者，网页版服务店员和管理员。店员和管理员共用网页版，登录后根据角色跳转对应页面。

### 权限矩阵

| 功能模块 | 消费者(C) 小程序 | 店员(S) 网页版 | 管理员(A) 网页版 |
|---------|-----------------|---------------|-----------------|
| 浏览菜单 | 可访问 | - | - |
| 下单 | 可操作 | - | - |
| 查看自己的订单 | 可访问 | - | - |
| 查看全部订单 | - | 可访问 | 可访问 |
| 处理订单(接单/制作/完成) | - | 可操作 | 可访问 |

| 配置门店信息 | - | - | 可操作 |
| 管理店员账号 | - | - | 可操作 |

## 3. MVP 范围界定

### IN scope

- 微信授权登录（C端，小程序）
- 账号密码登录（S端/A端，网页版）
- 菜单浏览（分类导航 + 商品列表 + 商品详情）
- 购物车
- 下单（确认订单后直接提交，不接入支付）
- 订单状态追踪
- 历史订单 + 再来一单
- 店员处理订单（接单 → 制作 → 待取餐 → 已完成）
- 管理员管理商品/分类
- 管理员配置门店信息（营业时间、公告等）
- 订单状态变更通知

### OUT scope（本次不做，记录为后续迭代）

- 在线支付（微信支付等）
- 多门店选择
- 会员积分/等级
- 优惠券/促销活动
- 外卖配送
- 商品定制选项（温度/甜度/奶类等）
- 精品咖啡特色字段（豆子信息、冲煮方式）
- 商品搜索
- 数据统计报表
- 微信订阅消息

## 4. 全局交互规范

### 加载状态

- 页面首次请求数据且本地无缓存时：展示骨架屏
- 操作反馈（提交订单等）：使用 loading 按钮（按钮内显示 loading 动画）
- 下拉刷新：使用小程序原生下拉刷新

### 空状态

- 购物车为空：展示空状态插图 + 文案「购物车是空的，去点杯咖啡吧」+ 跳转菜单按钮
- 订单为空：展示空状态插图 + 文案「还没有订单记录」
- 商品列表为空：展示文案「该分类暂无商品」

### 错误提示

| 场景 | 提示方式 | 示例 |
|------|---------|------|
| 网络异常 | Toast | 「网络不给力，请重试」 |
| 操作失败 | Toast | 「下单失败，请重试」 |
| 权限不足 | Toast | 「无权访问」 |
| 表单校验不通过 | 行内提示 | 输入框下方红字 |

### 下拉刷新 / 上拉加载

- 订单列表：支持下拉刷新 + 上拉加载更多（分页）
- 菜单页：支持下拉刷新
- 其他列表页：同上

## 5. 全局导航结构

### C端导航（微信小程序）

TabBar 三个 Tab：

| Tab | 图标 | 页面 | 说明 |
|-----|------|------|------|
| 菜单 | coffee | 菜单主页 | 左侧分类 + 右侧商品列表 |
| 订单 | order | 订单列表 | 当前订单 + 历史订单 |
| 我的 | user | 个人中心 | 用户信息 + 退出登录 |

非 TabBar 页面：
- 商品详情页（从菜单页进入）
- 购物车页（从菜单页右上角图标进入）
- 确认订单页（从购物车进入）
- 订单详情页（从订单列表进入）

### S端导航（网页版）

店员登录后进入订单处理页，左侧侧边栏导航：

| 菜单项 | 页面 | 说明 |
|--------|------|------|
| 订单 | 订单处理 | Tab 切换：待处理/进行中/已完成 |

### A端导航（网页版）

管理员登录后进入管理后台，左侧侧边栏导航：

| 菜单项 | 页面 | 说明 |
|--------|------|------|
| 订单 | 订单列表 | 全量订单查看 |
| 商品 | 商品管理 | 分类 + 商品增删改查 |
| 门店 | 门店配置 | 基本信息/营业时间/公告 |
| 店员 | 店员管理 | 店员账号增删改 |
