# 菜单与商品

## 1. 业务目标

- C端：清晰浏览菜单，快速找到想喝的咖啡，顺畅完成选品加入购物车
- A端：灵活管理商品和分类，支持上下架控制

## 2. 数据实体

### 2.1 商品分类（Category）

| 属性 | 说明 | 示例 |
|------|------|------|
| 分类名称 | 分类的显示名 | 意式咖啡、手冲单品、特调饮品、无咖啡因、轻食甜点 |
| 图标 | 分类导航的图标标识 | 可选，MVP 可用文字替代 |
| 排序 | 分类的展示顺序 | 数字越小越靠前 |
| 是否启用 | 控制分类是否对 C端可见 | 启用/禁用 |

### 2.2 商品（Product）

| 属性 | 说明 | 示例 |
|------|------|------|
| 所属分类 | 关联到具体分类 | 意式咖啡 |
| 商品名称 | 主标题 | 燕麦拿铁 |
| 副标题 | 一句话亮点 | 燕麦奶与浓缩的经典搭配 |
| 描述 | 详细介绍（可多行） | 产地、口感等信息 |
| 基础价格 | 商品标价（单位：元） | 28.00 |
| 商品图片 | 主图，1 张 | URL |
| 是否上架 | 控制商品是否可购买 | 上架/下架 |
| 排序 | 同分类下的展示顺序 | 数字越小越靠前 |

## 3. C端 — 菜单浏览

### 3.1 菜单主页

页面布局：左侧分类导航 + 右侧商品列表

**左侧分类导航**：
- 纵向排列所有启用的分类
- 当前选中分类高亮
- 点击切换右侧商品列表
- 分类过多时支持滚动

**右侧商品列表**：
- 按分类下的排序展示商品
- 每个商品卡片展示：图片、名称、副标题、基础价格
- 已下架商品不在列表中展示
- 商品卡片点击 → 进入商品详情页

**页面顶部**：
- 营业状态标识（营业中/已打烊）
- 门店公告滚动展示（有公告时展示，无公告时隐藏）

**页面右上角**：
- 购物车图标（带数量角标，显示购物车内商品件数）
- 点击进入购物车页

**登录引导**：
- 未登录用户可浏览菜单和商品详情
- 商品详情页的「加入购物车」按钮对未登录用户点击后，先引导登录，登录成功后自动回到该商品详情页

### 3.2 商品详情页

- 顶部：商品大图
- 商品名称 + 副标题
- 商品描述（如有）
- 价格
- 数量选择器（-/数字/+）
- 「加入购物车」按钮（显示当前数量 × 单价的总金额）

### 3.3 购物车

**购物车数据**：保存在本地，关闭小程序后仍保留，但不跨设备同步。

**购物车列表**：
- 每项展示：商品名称、单价、数量、小计
- 数量可修改（+/-）
- 数量减至 0 时自动移除该项
- 支持左滑删除

**底部结算栏**：
- 左侧：合计金额
- 右侧：「去结算」按钮
- 「继续购物」文字链接，返回菜单页
- 购物车为空时，结算栏不显示，展示空状态

**营业状态校验**：
- 门店打烊时，购物车页面提示「门店已打烊，暂不可下单」
- 「去结算」按钮置灰不可点击

### 3.4 营业状态变化感知

- 用户停留在菜单页或购物车页时，营业状态变化（如门店打烊）应在短时间内生效
- 打烊时：菜单页商品卡片不展示「加入购物车」入口，购物车页「去结算」按钮置灰
- 恢复营业时：自动恢复可操作状态

## 4. A端 — 商品管理（网页版）

### 4.1 分类管理

**分类列表**：
- 展示所有分类（包含启用和禁用）
- 每项展示：分类名称、排序、启用状态
- 支持拖拽排序（P1）

**新增/编辑分类**：
- 分类名称（必填）
- 图标（选填）
- 排序（数字输入，默认为当前最大值+1）
- 启用状态（开关，默认开启）

**删除分类**：
- 分类下有商品时不可删除，提示「请先移除该分类下的商品」
- 分类下无商品时可删除，二次确认

### 4.2 商品管理

**商品列表**：
- 默认展示全部商品
- 支持按分类筛选（下拉选择）
- 每项展示：商品图片（缩略图）、名称、分类、价格、上架状态
- 上架/下架切换（开关或按钮）

**新增商品**：
- 所属分类（下拉选择，必填）
- 商品名称（必填）
- 副标题（选填）
- 描述（选填，多行文本）
- 基础价格（必填，数字，精确到分）
- 商品图片（必填，1 张，支持上传）
- 排序（数字输入，默认为当前分类下最大值+1）
- 上架状态（开关，默认开启）

**编辑商品**：
- 与新增相同的表单，预填现有数据

**删除商品**：
- 二次确认「确定删除该商品？」
- 删除后不可恢复
- 已有订单中引用该商品的信息不受影响（订单保存商品快照）

**上下架**：
- 下架：商品对 C端不可见
- 上架：商品对 C端可见
- 支持在列表中直接切换
